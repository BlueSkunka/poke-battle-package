# PokeBattle Package

This package aims to share code between PokeBattle React app 
and PokeBattle Api.

## Build and publish
Upgrade the `package.json` version property.

Build with : 
```bash
npm run build
```

Publish with :
```bash
npm run publish
```

Then go in both app and update the package with : 
```bash
yarn install @blueskunka/poke-battle-package
```
